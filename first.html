PROJECTS

Amazon clone

This project is a clone of e-commerce web application inspired by Amazon. The platform replicates core features of an online shopping website, providing users with a seamless and intuitive experience for browsing, purchasing, and managing orders. It was built as part of a practical learning experience in web development, showcasing skills in frontend and backend technologies.

    Key Features:

  • User Authentication: Secure user login and signup with password encryption.
  • Product Listings: Dynamic display of products with filtering and search functionality.
  • Product Details: Detailed product pages with descriptions, pricing, and ratings.
  • Shopping Cart: Add-to-cart functionality with the ability to modify quantities or remove items.
  • Order Placement: Secure checkout process integrated with a payment gateway.
  • Order History: Users can view their past orders and track delivery status.
  • Responsive Design: Fully responsive UI for an optimal experience across devices.
